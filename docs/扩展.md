# k-Coder 扩展配置

## 文件位置与优先级

运行时按以下顺序加载指令，后加载的内容优先级更高：

1. 应用数据目录 `runtime-data/AGENTS.md`
2. 工作区根目录 `AGENTS.md`
3. 工作区 `.k-coder/rules/*.md`，按文件名排序

扩展配置按全局、项目顺序合并；相同 ID 由项目配置覆盖：

- 应用数据目录 `runtime-data/extensions.json`
- 工作区 `.k-coder/extensions.json`

已存在的文件如果不是 UTF-8、超过限制或格式错误，扩展准备和后续 Turn 会关闭失败。

## Skills

Skill 放在 `runtime-data/skills/<name>/SKILL.md` 或 `.k-coder/skills/<name>/SKILL.md`。项目同名 Skill 覆盖全局版本。

```markdown
---
name: review
description: Review repository changes
triggers:
  - review
  - code review
risk: read
enabled: true
---
Read the affected files and report findings by severity.
```

`name` 必须与目录名一致并使用小写 kebab-case。`risk` 只能是 `read`、`write`、`delete` 或 `external`。非 `read` Skill 即使声明 `enabled: true` 也默认停用，必须在设置中心显式启用。Skill 指令不会改变工具策略。

## MCP

stdio 示例：

```json
{
  "mcpServers": [
    {
      "id": "github",
      "enabled": true,
      "timeoutMs": 30000,
      "transport": "stdio",
      "command": ["node", "server.mjs"],
      "secret_env": {
        "GITHUB_TOKEN": "github-token"
      }
    }
  ],
  "hooks": []
}
```

Streamable HTTP 示例：

```json
{
  "mcpServers": [
    {
      "id": "remote",
      "transport": "streamable_http",
      "url": "https://example.com/mcp",
      "secret_headers": {
        "Authorization": "remote-authorization"
      }
    }
  ]
}
```

`secret_env` 和 `secret_headers` 的值是凭据名称，不是密钥。密钥通过设置中心写入系统凭据存储。远程 HTTP 必须使用 HTTPS；只有 loopback 地址允许 HTTP。工具以 `mcp__<server>__<tool>` 暴露。

## Hooks

```json
{
  "mcpServers": [],
  "hooks": [
    {
      "id": "external-guard",
      "phase": "before",
      "tool": "mcp__github__*",
      "command": ["node", ".k-coder/hooks/guard.mjs"],
      "timeoutMs": 10000,
      "enabled": true
    }
  ]
}
```

`phase` 为 `before`、`after` 或 `both`。Hook 从 stdin 读取单个 JSON 对象，并向 stdout 输出不超过 64 KiB 的单个 JSON 对象：

```json
{ "decision": "allow", "message": "", "output": null }
```

`decision: block` 会阻止工具。后置 Hook 可以用 `output` 替换返回给模型的文本。超时、非零退出、无效 JSON 和未知字段都会阻止工具。

## 诊断与审计

设置中心显示指令来源、Skill 风险、MCP 状态、工具数量、凭据状态、Hooks 和最近审计记录。审计保存在 `runtime-data/extension-audit.jsonl`，不包含密钥值。
